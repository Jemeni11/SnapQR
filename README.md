## Introduction

SnapQR is a QR code scanner and generator app for mobile devices. It is built with React Native and Expo.

## Planned Features

- [ ] Scan QR codes
- [ ] Generate QR codes
- [ ] Save QR codes
- [ ] Share QR codes
- [ ] Scan history
- [ ] Scan statistics
- [ ] Scan security
- [ ] Cloud Backup
